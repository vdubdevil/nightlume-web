Support & FAQ

Version

Which Minecraft version
is supported?

Currently, Nightlume Visuals supports Minecraft 1.16.5. Nightlume is distributed with a simple open-source launcher that automatically launches Minecraft with all required libraries.

Linux

How do I install
it on Linux?

For Linux environments, you can simply use the provided shell script from our GitHub repository or the "Downloads" page. Launch the launcher through any terminal environment. No complex installers required — just a simple launcher.

Technical

Is it a standalone
client or a mod?

Nightlume Visuals is a console-launcher client. Unlike standard mods, it does not rely on a traditional mods folder. The launcher creates its own isolated JVM environment at runtime for improved compatibility and stability.

Performance

Will it decrease
my FPS?

On the contrary. Nightlume uses a custom-built Void Rendering Pipeline. By bypassing standard Minecraft rendering bottlenecks and using optimized ASM-hooks, we achieve smoother frame timings and a more responsive UI compared to traditional mods.

Updates

How often is
the software updated?

We follow a "Rolling Release" model. Critical stability patches are pushed instantly to the GitHub repository, while major visual overhauls are released every 2-3 weeks following the roadmap in our Telegram channel.

Security

Is it safe
for multiplayer?

Our software is built with a focus on Privacy and Integrity. Nightlume contains no offensive modules or cheat-logic. It is a visual debugger and aesthetic enhancement tool, fully compliant with any server that allows custom client-side visualizers.
